Machine Learning

Machine Learning (ML) is a type of Artificial Intelligence (AI). Machine learning is literally what it means, a machine learns from the input given by a human. This data (input) can pretty much be anything. From games to loads of text, media and everything in between.
Machine learning algorithms predict outcomes from given data. Machine learning is mostly used in speech recognition, e-mail filtering, etc.
